# Changelog

## 2026-04-23

For a few days, Rangle has been available as a [Discord activity](https://discord.com/oauth2/authorize?client_id=1495567479978725476).

I've now added a server leaderboard to this!

(Note that you may be asked to authorise the activity again, now with the new permission to view what servers you're in)

## 2026-04-18

Cloud sync is now available for invited users.

## 2026-04-14

Added some more sound effects for gameplay. You can also now toggle sound effects on and off in the settings menu.

Also added customised browser icons for each theme, and gave a couple themes some custom cursors. Hope you like them :)

## 2026-04-08

Today was the first puzzle with a **bonus round!**
Those who solve the Rangle puzzle will be able to attempt a bonus round for extra points.
In the bonus round, players guess one or more of the values that hasn't been revealed. If you're within 5%, you get bonus points (and bragging rights).

## 2026-04-04

Themes! Now available in the settings cog. Some are freely available, but some you'll have to win!

## 2026-04-02

Hardcore mode is now available!
If you want to test your skills, hardcore mode will not reveal which stats are in the correct position, only the number which are correct.
Be careful, once you make a guess, the toggle is locked!
