# Changelog

## 2026-04-08

Today was the first puzzle with a **bonus round!**
Those who solve the Rangle puzzle will be able to attempt a bonus round for extra points.
In the bonus round, players guess one or more of the values that hasn't been revealed. If you're within 5%, you get bonus points (and bragging rights).

## 2026-04-04

Themes! Now available in the settings cog. Some are freely available, but some you'll have to win!

## 2026-04-02

Hardcore mode is now available!
If you want to test your skills, hardcore mode will not reveal which stats are in the correct position, only the number which are correct.
Be careful, once you make a guess, the toggle is locked!
